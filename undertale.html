FAQ

Updated: 11/09/2020.


Are you working with Toby Fox? This project is somewhat related to UNDERTALE/DELTARUNE development?

In short: No.

This project is volunteer-searching?

Yes! You can contact me within GameJolt or Discord.

Why the fan-game is named "Undertale: Red"?

Long short story, the original name of the fan-game was "Undertale: Determination", but I decided to change it because that name was already picked.

So, I changed the name to Undertale: Red, because red is the color that represents Determination.

What features will have this game?

I don't want you to have high expectations. I am not actually a experienced game programmer, and I am making this fan-game for hobby.

I am actually trying to make some features that make Undertale be Undertale.

I'm trying my best.

The project has a team?

No. I'm working alone.

Will the game follow the Undertale history?

If we start to think, I think it would be boring to the player to only follow a path. I think you should have decisions in the game that affects the character dialogs. I'm using the Undertale storyline, but that doesn't mean it's just a route.

In percentages, how much content is in the game?

The actual version is only a prototype game. The menu, splash, and other type of systems are done, and are having minimal improvements. The actual version of the game is 0.0.0.2, because the dialogs on the overworld are being created, the player movement is functional. So, the game is just 0.0002% completed.

I don't even know if that's mathematically accurate, I just want you to understand it's freakin' low.

What was the inspiration for creating the game?

I got inspired by Undertale Yellow and Green for creating a same history with a previous character.
